# Student Grade System (Python)
##  Description
This is a beginner-friendly Python project that calculates a student's **grade** and **pass/fail result** based on the marks entered by the user.

The program runs in the command prompt and uses simple conditional logic.

##  Concepts Used
- User Input (`input()`)
- Data Type Conversion (`float`)
- Conditional Statements (`if-elif-else`)
- Comparison Operators (`>=`, `<`)
- Console Output Formatting

## How to Run the Project
1. Open Command Prompt
2. Navigate to the project folder:
   cd "C:\github projects\student-grade-system"
3. Run the program:
   py main.py
